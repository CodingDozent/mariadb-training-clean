# MariaDB Trainingsumgebung (GitHub Codespaces)

## Voraussetzungen
- GitHub-Account
- Zugriff auf dieses Repository

## Start

1. Repository auf GitHub öffnen.
2. Auf den grünen Button **Code** klicken.
3. Reiter **Codespaces** → **Create codespace on main**.
4. Warten, bis der Codespace gestartet ist (ca. 2-3 Minuten).

## Dienste

- MariaDB:
  - Host: 127.0.0.1
  - Port: 3306
  - Benutzer: root
  - Passwort: root
- phpMyAdmin:
  - Port 8888 (wird in Codespaces automatisch verlinkt)
  - Login: root / root

## Beispiel-Datenbank

- Name: `schule`
- Tabelle: `personen`

Die Aufgaben findet ihr im Skript / Aufgabenblatt.
